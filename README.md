# Функциональный мессенджер с использованием Qt, C++ и DB

### Это дипломный проект для итоговой аттестации по курсу "С++ разработчик" в онлайн-школе [SkillFactory](https://skillfactory.ru/) студента [Родиона Королева](https://github.com/jminora)

### Реализованы следующие функции:
* GUI для серверной части приложения;
* Серверная часть приложения позволяет просматривать все сообщения (в том числе приватные);
* Серверная часть приложения позволяет просматривать список пользователей;
* Cерверная часть приложения позволяет отключать и банить пользователей;
* Внесены изменения и дополнения к интерфейсу клиентской части (добавлена система уведомлений с табами и звуковыми сигналами).
